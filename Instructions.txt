1. git init- Used to create any repository a git repository
2. git clone "link from github"- clones github repository to your local machine.
3. git status- shows all the modified files.(and their commit status)
               shows all the files that are not yet tracked by git.
4.git add (fileName)- adds a particular file to get to get tracked.
           git add .    - adds all the files(modified and untracked) to the git.
5. git commit- save your changes to the git in your local machine.
   format: git commit -m "Title" -m "Description".
6. git push - save your changes to the server e.g. Github.
   for Git push you need to connect your local machine to your Github Account.



Branching:
1. Git branch: will tell all the local branches and the branch on which you are working.
2. Git branch <branchName>: Creates a new local branch
3. Git branch -v: Lists all local and remote branches.
4. Git checkout <branchName>: Switches to branchName.
5. Git branch -d <branchName>: Deletes a branch locally (you should not be on the branch that you are deleting)
